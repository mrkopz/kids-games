# kids-games

เกมสำหรับเด็ก เล่นบนเบราว์เซอร์ ไม่ต้องติดตั้งอะไรเพิ่ม เปิดไฟล์ HTML แล้วเล่นได้เลย

## หน้าเมนูรวมเกม ([`index.html`](index.html))

หน้าแรกที่รวมเกมทั้งหมด แตะการ์ดเกมที่อยากเล่นแล้วจะพาไปหน้าเกมนั้นเลย เปิด `index.html` เป็นจุดเริ่มต้น

## เอาขึ้น Netlify

ทั้งโปรเจกต์เป็นไฟล์ static ล้วน ไม่ต้อง build:

- **ลากวาง:** เปิด [app.netlify.com/drop](https://app.netlify.com/drop) แล้วลากโฟลเดอร์นี้ทั้งโฟลเดอร์เข้าไป
- **เชื่อม Git:** เชื่อม repo กับ Netlify ได้เลย ไม่ต้องตั้ง build command — มี `netlify.toml` กำหนด `publish = "."` ให้แล้ว Netlify จะเสิร์ฟ `index.html` เป็นหน้าแรกอัตโนมัติ

## เกมที่มี

### 🎴 เกมจับคู่ ([`memory-games.html`](memory-games.html))

เกมจับคู่ภาพสัตว์ พลิกการ์ดหาคู่ที่เหมือนกัน เหมาะกับเด็กเล็ก เล่นได้ทั้งคนเดียวและสองคน

**ฟีเจอร์**
- 👤 โหมด 1 คน (ฝึกความจำ) หรือ 2 คน (แข่งกัน)
- 📊 3 ระดับความยาก: ง่าย (6 คู่) / กลาง (8 คู่) / ยาก (10 คู่)
- ⏱ จับเวลาต่อรอบในโหมด 2 คน (15s / 30s / 45s หรือไม่จำกัด) ถ้าหมดเวลาเปลี่ยนคน
- 🔊 เสียงพูดชื่อสัตว์เป็นภาษาอังกฤษเมื่อพลิกการ์ด (ลูกได้เรียนคำศัพท์ไปด้วย) — ใช้ Web Speech API ของเบราว์เซอร์
- 🎵 เสียงเอฟเฟกต์สังเคราะห์ด้วย Web Audio API ไม่ต้องโหลดไฟล์เสียง
- 📱 ออกแบบสำหรับ iPad รองรับทั้ง portrait/landscape ปรับ grid ของการ์ดให้พอดีหน้าจออัตโนมัติ
- 🎉 มี confetti กับเสียงดนตรีตอนชนะ

**วิธีเล่น**
1. เปิดไฟล์ `memory-games.html` ในเบราว์เซอร์ (Safari, Chrome ก็ได้)
2. กด ⚙️ ตั้งค่า เพื่อเลือกจำนวนผู้เล่น/ความยาก/เวลาต่อรอบ
3. แตะการ์ดเพื่อพลิกหาคู่ จับคู่ถูก = ได้เล่นต่อ, ผิด = สลับผู้เล่น (ในโหมด 2 คน)

### 🔤 เกมสะกดชื่อสัตว์ ([`spelling-game.html`](spelling-game.html))

โชว์รูปสัตว์แล้วให้เด็กสะกดชื่อ ฝึกอ่าน/สะกดคำได้ทั้งภาษาไทยและอังกฤษ

**ฟีเจอร์**
- 🇹🇭🇬🇧 สลับภาษาไทย / อังกฤษ ได้
- 🎮 3 โหมดเล่น ปรับได้:
  - **เรียงทีละตัว** — แตะตัวอักษรเรียงจนครบคำ (มีตัวลวงปน)
  - **เติมตัวที่หาย** — คำมีช่องว่าง 1 ช่อง เลือกตัวที่ถูกจาก 3 ตัว
  - **เลือกคำที่ถูก** — เลือกชื่อสัตว์ที่ถูกจาก 3 คำ
- 👤 โหมด 1 คน หรือ 2 คน (สลับกันตอบทีละข้อ นับคะแนนแยก)
- 📊 เลือกจำนวนข้อได้: 5 / 10 / 15
- 🔊 เสียงพูดชื่อสัตว์เมื่อตอบถูก รองรับทั้งไทย (`th-TH`) และอังกฤษ (`en-US`) — Web Speech API
- 🎵 เสียงเอฟเฟกต์ถูก/ผิด สังเคราะห์ด้วย Web Audio API ไม่ต้องโหลดไฟล์เสียง
- 📱 ออกแบบสำหรับ iPad กันจอซูมเวลาเด็กกดเร็วๆ
- 🎉 มี confetti ตอนตอบถูกและตอนจบเกม

**วิธีเล่น**
1. เปิดไฟล์ `spelling-game.html` ในเบราว์เซอร์ (Safari, Chrome ก็ได้)
2. กด ⚙️ ตั้งค่า เพื่อเลือกภาษา / โหมด / จำนวนผู้เล่น / จำนวนข้อ
3. ดูรูปสัตว์แล้วแตะตัวอักษรหรือคำให้ถูก ตอบถูกได้คะแนน เล่นจนครบทุกข้อ

## การพัฒนา

ไม่ต้อง build หรือ install dependency อะไรเลย ทุกอย่างเป็น HTML/CSS/JS ในไฟล์เดียว แก้แล้ว refresh ได้เลย

- เกมจับคู่: แก้ `EMOJI_POOL` และ `EMOJI_NAMES` ใน `memory-games.html` เพื่อเปลี่ยนสัตว์
- เกมสะกดคำ: แก้คลัง `ANIMALS` ใน `spelling-game.html` (มี `emoji`, `th`, `en` ต่อสัตว์ 1 ตัว) เพื่อเพิ่ม/เปลี่ยนสัตว์
